# Install

```
git clone git@github.com:paolodina/fastapi-users-ormar-demo.git
cd fastapi-users-ormar-demo
poetry shell
poetry install
python create_db.py
uvicorn main:app
```

# Postman

It's available a postman collection `FastAPI-Users-ormar.postman_collection.json`
you can import to do some test.
